## 📌 Project Overview

This project demonstrates a cybersecurity risk management framework applied to a healthcare-style organizational environment.

The objective is to:

✔ Identify security risks  
✔ Assess vulnerabilities  
✔ Develop mitigation strategies  
✔ Enhance security governance  
✔ Support business continuity

The project aligns with industry risk management principles and cybersecurity best practices.

---

## 🛠 Tools and Technologies Used

Hands-on security analysis and labs were performed using:

| Tool | Purpose |
|------|----------|
| Nmap | Network scanning and host discovery |
| nbtscan | NetBIOS device discovery |
| SimpleRisk | Risk management documentation |
| GVM | Vulnerability scanning and analysis |
| FTP | Data exposure and security analysis |
| Metasploit | Exploitation framework for security testing |

These tools provided practical experience in:

✔ Vulnerability assessment  
✔ Network security analysis  
✔ Risk documentation  
✔ Security testing
